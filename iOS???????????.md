# 出口成章｜iOS 上架开发计划与任务拆解

## 0. 总目标
把“出口成章”从概念与 Demo，推进到：
1. iPhone 上可运行
2. 核心主流程跑通
3. 可进行一轮真实用户自测
4. 可提交 TestFlight
5. 满足 App Store 上架基础要求

---

## 1. 产品目标定义

### 1.1 首版要解决的问题
- 用户能把一段语音表达录进去
- 系统能转成文字
- 系统能输出“优化稿”
- 用户能做第 2 遍复述训练
- 系统能给出训练反馈
- 内容能归档到灵感库
- 用户能看到每日表达报告

### 1.2 首版不做
- 团队协作
- 社交分享社区
- 复杂知识图谱
- 多端同步的复杂权限系统
- 过重的增长玩法

---

## 2. 上架版功能范围

### P0 必做
1. 登录 / 本地身份识别（最小可先匿名本地）
2. 录音
3. 音频上传
4. 语音转文字
5. 原文 / 优化稿展示
6. 表达问题分析
7. 训练模式（至少支持 2 轮）
8. 灵感归档
9. 历史记录
10. 每日报告

### P1 应做
1. 多种优化模式（简洁 / 正式 / 汇报 / 口播）
2. 标签管理
3. 搜索与筛选
4. 训练结果趋势
5. 产品灵感分类建议

### P2 可后置
1. 动态灵感网络图
2. 云端同步
3. 订阅付费
4. 团队版
5. 多语言支持

## 3. iOS App 页面任务拆解

## 3.1 首页
### 目标
让用户 3 秒内知道怎么开始。

### 任务
- [ ] 顶部品牌区：产品名 + slogan
- [ ] 主按钮：开始录音
- [ ] 最近记录卡片
- [ ] 今日报告概览卡
- [ ] 快捷入口：训练 / 灵感库 / 报告

### 验收
- [ ] 首页首屏能直接开始录音
- [ ] 能看到最近记录
- [ ] 能跳到各主要页面

---

## 3.2 录音页
### 目标
用户可以顺畅录入一段表达。

### 任务
- [ ] 开始/暂停/结束录音
- [ ] 录音时长显示
- [ ] 权限请求（麦克风）
- [ ] 场景选择（汇报 / 面试 / 口播 / 灵感 / 通用）
- [ ] 音频文件保存
- [ ] 录音后发起转写

### 验收
- [ ] 真机可录音
- [ ] 权限请求正常
- [ ] 录完能进入转写流程

---

## 3.3 结果页
### 目标
让用户明确看见“变好了”。

### 任务
- [ ] 原文卡片
- [ ] 优化稿卡片
- [ ] 差异高亮（可先简化）
- [ ] 问题分析区
- [ ] 优化模式切换
- [ ] CTA：开始训练 / 归档灵感

### 验收
- [ ] 原文和优化稿都能显示
- [ ] 用户能看懂改了什么
- [ ] 可以进入训练和归档

---

## 3.4 训练页

### 任务

- [ ] 训练评分
- [ ] 反馈说明

### 验收
- [ ] 至少能完成 1 次复述训练
- [ ] 能看到评分和反馈
- [ ] 用户知道自己哪里提升了
